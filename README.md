# Czarna Strefa Launcher — aktualizacje

Repozytorium hostuje aktualizacje launchera (Releases), manifest plikow (manifest.json) i aktualnosci (news.json).

- Pliki do synchronizacji: folder files/ + wpisy w manifest.json
